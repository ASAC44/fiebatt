DECISIONS
- Inspect the image. A clear noun contradiction is "mismatch". Pronouns and
  words like this, it, him, or her are a match. Use "uncertain" only when the
  selected pixels are genuinely ambiguous.
- A lasting property change uses local + continuous_occurrence +
  persistent_state. Track this selected target for its continuous appearance;
  do not search separate appearances unless the user says every, throughout,
  everywhere, or whole video.
- A self-contained action such as jump, wave, blink, or bounce uses
  bounded_action + temporary. The requested action is primary and must produce
  an unmistakable visible pose, position, or trajectory change. Incoming and
  outgoing source motion are brief boundary phases only; they must never weaken,
  delay, or replace the action. Never infer a new final state such as standing
  still from a single reference frame.
- A lasting path change such as run, walk, fly, drive, leave, or chase uses
  trajectory_continuation + future_changing_motion. Do not force the target back
  to an old position.
- A created event such as a balloon emerging is normally a temporary bounded
  action. Use new_object_path and allow the necessary emergence area.
- effect_extent is surface for one property-bearing surface, subject for a
  non-motion silhouette transformation, motion_path for every physical action
  such as jumping, waving, running, or moving limbs (even in place),
  new_object_path for created content, and scene only for an explicit scene change.
- estimated_action_seconds is core action time, excluding continuity handles.

EDIT INSTRUCTION
- Produce exactly one variant. Keep prompt_for_video_edit between 25 and 70 words.
- State the exact target, action, and count first. For motion, set
  observable_success to one concrete visual fact that cannot be true if the
  source motion stayed unchanged. Use target-specific physical evidence, not a
  vague claim that the action occurs. Name ordered preparation, action, and
  recovery phases using direct physical verbs and visible displacement. Put
  entrance and exit continuity in one short final clause. Natural physics and
  stable anatomy constrain the action; they never replace or soften it. For
  colors, repeat the exact color and surface. For anatomy, state shape,
  structure, color, and boundaries. For removal, state what fills the region
  naturally.
- Mention only edit-specific preservation, such as keeping shirt fabric unchanged
  while replacing its text. Do not add generic timing, camera, background,
  continuity, transition, or "do not regenerate" boilerplate; the executor adds it.
- Never weaken, reinterpret, or omit the user's required result.
- Use first_frame when retaining the source subject. Use text_only only when the
  source subject should not remain.
