# Continuity

Continuity is one of the defining parts of the fiebatt workflow.

Once a user accepts a generated result, fiebatt can push that decision further through the surrounding cut so the edit feels authored instead of isolated.

## AI-driven Continuity Analysis

fiebatt now leverages AI scoring to analyze and ensure continuity. When an edit is accepted, the system can use endpoints like `/api/score/continuity` to evaluate how well the change integrates with surrounding frames and segments. This analysis helps in propagating the edit effectively and identifying potential inconsistencies.
