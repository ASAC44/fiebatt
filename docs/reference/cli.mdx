## Installation and Agent Setup

To install the fiebatt CLI and set up the agent skill for prompt-driven video editing, follow these steps:

1.  **Install the fiebatt CLI:**
    ```bash
    pip install fiebatt-edit
    ```
2.  **Set up the agent skill file:** This makes the fiebatt CLI available to AI agents (e.g., Claude).
    ```bash
    mkdir -p ~/.claude/skills/fiebatt-edit && curl -sL https://raw.githubusercontent.com/ASAC44/fiebatt/main/apps/cli/SKILL.md -o ~/.claude/skills/fiebatt-edit/SKILL.md
    ```
3.  **Authenticate with fiebatt:**
    ```bash
    fiebatt auth login --base-url https://fiebatt.local
    ```

Once installed and authenticated, you can start editing videos by saying "fiebatt edit" to your agent.
